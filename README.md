# PreZen

Discord VC 滞在時間管理システム

## プロジェクト概要

Discord のボイスチャンネルにおけるユーザーの滞在時間を自動記録し、Web 管理画面で可視化・管理するシステムです。教育機関での出席管理や、コミュニティの活動状況把握に活用できます。

## 主要機能

### Discord VC 監視機能

- 特定の VC チャンネルへの参加者を自動検出
- リアルタイムでの参加/退出イベントの記録
- 設定した時間帯での参加者カウント
- Bot 常時稼働、VC への物理接続なし(裏で監視)

### Web 管理画面

- **監視ステータス表示**: 現在の監視状態をリアルタイム表示
- **監視スケジュール設定**: 曜日・時間帯別の監視設定
- **ユーザー管理**: Discord ID との紐付け、所属課の管理
- **データ閲覧**: 滞在履歴、セッション別参加者一覧
- **データエクスポート**: CSV 形式でのダウンロード

### データ記録・分析

- ユーザーごとの滞在時間の累積記録
- 日時・チャンネル別の滞在履歴
- 入退出履歴の詳細記録
- 所属課別の統計分析(Phase 2 以降)

| **Container** | Docker + Docker Compose              |
